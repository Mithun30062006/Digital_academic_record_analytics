# Digital Academic Records - Backend

Setup:

1. Copy `.env.example` to `.env` and populate DB credentials.
2. Install dependencies:

```bash
npm install
```

3. Run in development:

```bash
npm run dev
```

Seeding local users (when using JSON storage):

```bash
npm run seed
```

Default seeded credentials:
- Admin: username `admin`, password `Admin@123`
- Student: student_number `S001`, password `Student@123`
